#### Sorting

```
 - Binary Search: O(log n)
 - Selection Sort: O(N²)
 - Bubble Sort: O(N²)
 - Insertion Sort: Best case = O(n) Average = O(N²) Worst Case = O(N²)
 - indexOf: O(n)
```
